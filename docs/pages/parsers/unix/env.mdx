Usage (cli)

```
env | rjc env
```

Schema

```
[
    {
        "name":  String,
        "value": String,
    }
]
```

Example

```json
$ env | rjc -o=pretty env
[
    ...
    {
        "name": "USER_BUILD",
        "value": "windows.1"
    },
    {
        "name": "USER_MAJOR",
        "value": "2"
    },
    {
        "name": "USER_MINOR",
        "value": "37"
    },
    {
        "name": "USER_PATCH",
        "value": "3"
    },
    {
        "name": "verbose_output",
        "value": "0"
    },
    {
        "name": "WINDIR",
        "value": "C:\\Windows"
    },
    ...
]
```
