Usage (cli)

```
wc * | rjc wc
```

Schema

```
{
    "meta": {
        "total_lines":      i32,
        "total_words":      i32,
        "total_characters": i32
    },
    "resources": [
        {
            "name":       String,
            "lines":      i32,
            "words":      i32,
            "characters": i32
        }
    ]
}
```
