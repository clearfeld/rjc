Usage (cli)

```
ls -al | rjc ls
```

Schema

```
{
    "resources" [
        {
            "name":         String,
            "permissions":  String,
            "links":        i32,
            "owner":        String,
            "group":        String,
            "size":         i32,
            "date":         String,
        }
    ]
}
```
