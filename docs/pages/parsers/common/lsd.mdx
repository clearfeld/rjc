Usage (cli)

```
lsd -al | rjc lsd
```

Schema

```
[
    "resources": {
        "name":        String,
        "permissions": String,
        "owner":       String,
        "group":       String,
        "date":        String,
        "size":        f32,
        "unit":        String,
    }
]
```
