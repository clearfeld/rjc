Usage (cli)

```
airport -I | rjc airport
```

Schema

```
{
    "agrCtlRSSI":      i32,
    "agrExtRSSI":      i32,
    "agrCtlNoise":     i32,
    "agrExtNoise":     i32,
    "state":           String,
    "op mode":         String,
    "lastTxRate":      i32,
    "maxRate":         i32,
    "lastAssocStatus": i32,
    "802.11 auth":     String,
    "link auth":       String,
    "BSSID":           String,
    "SSID":            String,
    "MCS":             i32,
    "guardInterval":   i32,
    "NSS":             i32,
    "channel":         String,
}
```
