Usage (cli)

```
dir | rjc dir
```

Schema

```
[
    "meta": {
        "drive":       String,
        "serial":      String,
        "directory":   String,
        "files":       i32,
        "directories": i32
    }
    "resources": [
        {
            "date":   String,
            "time":   String,
            "is_dir": bool,
            "size":   Option<i32>, // i32 | null
            "name":   String,
        }
    ]
]
```
