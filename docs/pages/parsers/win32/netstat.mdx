Usage (cli)

```
netstat -an | rjc netstat
```

Schema

```
{
    "tcp": [
        {
            "local_address":    String,
            "foreign_address":  String,
            "state":            String
        }
    ],
    "udp": [
        {
            "local_address":   String,
            "foreign_address": String
        }
    ]
}
```
